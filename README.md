# kwcs-msd-psychosocial

Replication code for:

**Trends in psychosocial risk factors and musculoskeletal disorders among Korean workers, 2010-2023: the autonomy paradox in a changing labour market**

Heo S, Choi T, Choi W. *BMC Public Health* (submitted).

## Overview

This repository contains the analysis code for a pooled cross-sectional study of 119,912 Korean workers from six waves (2010-2023) of the Korean Working Conditions Survey (KWCS). The study examines 13-year trends in psychosocial risk factors (Job Demand-Control-Support model) and musculoskeletal disorders (MSD), with attention to the "autonomy paradox" driven by occupational confounding.

## Data availability

KWCS microdata are publicly available from the Korea Occupational Safety and Health Agency (KOSHA):
https://oshri.kosha.or.kr

Due to data use agreements, raw data files are not included in this repository. Download the 2nd-7th wave CSV files from the KOSHA website and place them in the `data/` directory following the structure described below.

## Reproducing the analysis

### 1. Environment setup

```bash
pip install -r requirements.txt
```

### 2. Data preparation

Download KWCS waves 2-7 (2010-2023) from [KOSHA](https://oshri.kosha.or.kr) and place the CSV/DAT files in the `data/` directory. See `data/README.md` for the expected file structure.

### 3. Run notebooks sequentially

```
0_extract.ipynb      → outputs kwcs_pooled_raw.csv (N=260,996)
1_preprocess.ipynb   → outputs analytic_sample.csv (N=119,912)
2_analysis.ipynb     → outputs Tables 1-3, Figures 1-4, Tables S1-S3 (regression N=83,288)
3_supplementary_analyses.py → outputs Tables S4-S6 and robustness analyses
```

`3_supplementary_analyses.py` takes the path to `analytic_sample.csv` as an
optional first argument. It re-derives every supplementary estimate reported
in the manuscript — wave-stratified prevalence ratios, exposure-by-wave
interactions, the support-omitted and multiply-imputed (m=10, Rubin's rules)
models, age-stratified estimates, total vs direct effect, wave-clustered
variance, RERI with delta-method and 20,000-draw parametric-bootstrap CIs,
and the E-value — and writes `verification_report.txt` comparing each
reproduced value against the published value (93 checks).

```bash
python notebooks/3_supplementary_analyses.py output/pre_output/analytic_sample.csv
```

Each notebook can be run in Google Colab or any Jupyter environment with the required packages.

## Methods summary

| Component | Detail |
|-----------|--------|
| Design | Pooled repeated cross-sectional (6 waves, 2010-2023) |
| Sample | N=119,912 Korean workers (KWCS waves 2-7) |
| Exposure | JDC-S: psychological demand, decision latitude, social support |
| Outcome | Self-reported MSD (back, upper limb, lower limb) |
| Analysis | Modified Poisson regression with robust SE (Zou, 2004) |
| Interaction | Autonomy x ergonomic exposure (RERI) |
| Sensitivity | 5 analyses + E-values |
| Software | Python 3.10, statsmodels 0.14, pandas, numpy, matplotlib |

## Key findings

1. Low decision latitude nearly doubled (34% to 64%) while long working hours declined (46% to 15%)
2. Demand and support associated with MSD as predicted by JDC-S theory
3. Decision latitude showed a robust, paradoxical inverse association with MSD (PR 0.84, 95% CI 0.83-0.86) that is unchanged when industry and employment type are removed and persists within every large industry - a non-causal signal more plausibly reflecting the decision-latitude measure, healthy-worker selection, and reverse causation than a protective effect of low control
